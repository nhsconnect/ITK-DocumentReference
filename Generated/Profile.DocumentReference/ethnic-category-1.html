    Ethnic Category

    Summary

    Name Description
    Defining URL: https://fhir.nhs.uk/ValueSet/ethnic-category-1
    Name: Ethnic Category
    Definition: The vocabulary describes the ethnic category which the patient regards as their own, designed to be used with later versions of PDS. It is an extension of the ETHNIC CATEGORY CODE described in the NHS Data Model and Dictionary for England.
    OID: 2.16.840.1.113883.2.1.3.2.4.16.6
    Status: Active
    Last Updated: 03/08/2016 00:00:00 +01:00
