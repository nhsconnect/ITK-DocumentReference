    Summary

    Name Description
    Defining URL: https://fhir.nhs.uk/ValueSet/cda-person-relationship-type-1
    Name: CDA Person Relationship Type
    Definition: A relationship between one person and another. These relationships are not necessarily mutually exclusive, for example a parent may be a child's next of kin. Includes the set of values defined in e-GIF Person Relationship Type
