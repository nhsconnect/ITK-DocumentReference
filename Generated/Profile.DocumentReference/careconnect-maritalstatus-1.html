    Care Connect Marital Status

    Summary

    Name Description
    Defining URL: https://fhir.hl7.org.uk/ValueSet/CareConnect-MaritalStatus-1
    Name: Care Connect Marital Status
    Definition: This value set defines the set of codes that can be used to indicate the marital status of a person.
    OID: 2.16.840.1.113883.4.642.2.19
    Status: Active
    Last Updated: 03/08/2016 00:00:00 +01:00
    Source Resource:

    XML

    The codes listed in this FHIR ValueSet are a restricted set of the FHIR Marital Status set of codes. This FHIR ValueSet has a 'Required' binding; for conformance the Marital Status element SHALL include a code listed below.


    This valueset includes all the codes for Care Connect Marital Status ( http://hl7.org/fhir/marital-status )

    Code Display Definition
    U

    Unmarried

    		The person is not presently married. The marital history is not known or stated.

    In addition, this value set includes codes from other code systems.

    Include these codes as defined in http://hl7.org/fhir/v3/MaritalStatus :

    Code Display
    D

    Divorced
    L

    Legally Separated
    M

    Married
    S

    Never Married
    W

    Widowed

    Include these codes as defined in http://hl7.org/fhir/v3/NullFlavor :

    Code Display
    UNK

    unknown

    This valueset includes a mapping to a seperate valueset

    Code Display Mapping
    U Unmarried Equivalent~S
    D Divorced Equivalent~D
    L Legally Separated Equivalent~P
    M Married Equivalent~M
    S Never Married Equivalent~S
    W Widowed Equivalent~W
    UNK unknown Equivalent~N
